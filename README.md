# 🎬 Netflix Recommendation System

AI-powered movie recommendation system built using Python, NLP, Scikit-learn, and Streamlit.  
This project recommends similar movies based on content similarity and metadata from the TMDB dataset.

---

# 🚀 Features

- Movie recommendation engine
- Content-based filtering
- Similar movie suggestions
- Interactive Streamlit web app
- Clean and responsive UI
- TMDB dataset integration

---

# 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- NLTK
